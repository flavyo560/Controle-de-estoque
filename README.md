# DEKIDS - Sistema de Gestao

Sistema de gestao de estoque e vendas com Flet e Supabase.

## Deploy no Render

1. Crie Web Service no Render
2. Conecte o repositorio GitHub
3. Configure variaveis de ambiente:
   - SUPABASE_URL
   - SUPABASE_KEY
   - PORT=8000
4. Deploy!
